How to Contribute

Contributions are welcome! Don't hesitate to contact us if you

  • found a bug;
  • have a suggestion on how to improve the code and/or documentation;
  • would like to get involved in writing code and/or documentation.

You can contact us by raising an issue on Github, by writing to one of the developers or at shareloqs@framagroupes.org.

We have contribution guidelines and a community code of conduct.
